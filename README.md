CoreCraft — это сверхлегковесная, быстрая и портативная среда разработки (IDE) для языка программирования Python, созданная разработчиками из Vortex Core Labs. 

Если вам надоел тяжелый PyCharm, а стандартный IDLE кажется неудобным — CoreCraft предлагает минималистичный интерфейс для быстрого написания и запуска вашего Python-кода в один клик.

✨ Ключевые особенности
Легковесность: Запускается мгновенно и не нагружает оперативную память.
Портативность (Portable): Программа работает без установки. Просто скачайте и запускайте.
Всё в одном: Напишите код в верхнем окне и сразу запускайте его через удобное встроенное меню.
Чистый оффлайн: IDE работает локально, не собирает и не отправляет ваши личные данные в сеть.

📦 Как использовать
1. Скачайте файл `CoreCraft.exe`.
2. Запустите программу (установка не требуется).
3. Введите ваш Python-код в поле редактора.
4. Запустите код и смотрите результат!

Как использовать

1. Откройте программу
2. В белом окне сверху напишите Python код
3. Используйте меню для дополнительной информации
